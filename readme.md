## Project Overview

This project demonstrates an end-to-end DevOps + MLOps workflow for training, tracking, containerizing, and deploying a machine learning model that predicts house prices.

The pipeline includes dataset versioning, experiment tracking, API deployment, Docker containerization, CI/CD integration, and automated email alerts.

---

## Project Workflow

1. Dataset tracked using **DVC**
2. Model training tracked using **MLflow**
3. Model saved and served via **FastAPI**
4. Failure alerts sent via **email**
5. Docker containerization for deployment
6. GitHub Actions CI/CD pipeline for automation
